# Contributing to Alive

Thank you for your interest in contributing to Alive!

## Branch Protection Rules

To maintain the stability of the project, we have implemented the following rules:

1.  **Direct Pushes to `main` are Disabled:** You should not push directly to the `main` branch.
2.  **Pull Request Workflow:** All changes must be submitted via a Pull Request (PR).
